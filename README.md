# Swiggy-clone-food-delivery_app
This is a food delivery web app that works just like swiggy app.
People can browse restaurants, look at menus, order food and manage their shopping cart all in one place.
Techstacks used in this project are React, Redux, Tailwind Css and Parcel bundler.
It is easy to use and looks like a professional.
It works fast and does not crash.
It feels like using real swiggy app.
This is a complete  food delivery web app that shows we can build something that looks and works like real business app.
